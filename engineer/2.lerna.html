Leara介绍 ​

官网：https://lerna.js.org/

项目初始化 ​

mkdir han-cli
cd han-cli
npm i lerna -D

推荐大家全局也安装一个lerna，这样在任何地方都能使用lerna命令了，非常方便，要不然就得使用npx了，用npx lerna init去初始化。

npm i lerna -g

Lerna初始化 ​

Lerna init

在项目根目录下运行Lerna init命令，对Lerna进行初始化，运行后会在当前项目生成一个lerna.json、package.json和.gitignore文件。
