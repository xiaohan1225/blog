注意点 ​

  1. 注意大小写，搜索的时候应该忽略大小写

前端搜索核心代码 ​

js
const title = ''; // 搜索框输入的值
const allList = []; // 全部数据
// 根据输入的值，过滤出需要的数据
const newList = allList.filter(item => item.title.toUpperCase().indexOf(title.toUpperCase()) > -1);
