rollup使用 ​

命令行方式使用 ​

  1. 安装rollup
bash
npm i rollup -g

-g表示全局安装

  1. 新建index.js文件
js
// index.js
console.log('hello world');
  1. 使用rollup打包
bash
rollup index.js

配置文件方式使用 ​

js
// rollup.config.js 
export default {

}

rollup.config.js默认支持的是Es module，如果你需要用CommonJS方式书写配置文件，需要修改配置文件的名称，把rollup.config.js改成rollup.config.cjs。
