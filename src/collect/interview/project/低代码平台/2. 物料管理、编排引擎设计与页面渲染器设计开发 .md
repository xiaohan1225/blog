### 物料管理
#### 物料的定义
在低代码平台中，物料(Component)是用户界面(U)中构成页面或应用的基本元素。每个物料具有自己的类型（如文本、按钮、容器等）和相关属性。物料管理即是对这些组件的统一管理，确保组件的高效加载、显示和交互。

### 物料的状态管理
在 React 开发中，我们可以使用 Zustand等 状态管理库来管理物料的状态。Zustand 提供了一种简单的APl,用于管理应用的全局状态，可以高效地存储、修改和获取物料信息。

React 主流状态管理工具：
1. Redux
2. MobX
3. Zustand
4. Jotai

React需要缩小状态的订阅范围，提高性能：
