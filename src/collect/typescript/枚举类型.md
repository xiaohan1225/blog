## 数字枚举
使用`enum`关键词进行定义：
```ts
enum Color {
  Red, // 0
  Green, // 1
  Blue // 2
}
```

默认情况下，枚举成员会被赋值为从 `0` 开始递增的数字。

### 增长枚举
```ts
enum Color {
  Red = 1, // 1
  Green, // 2
  Blue // 3
}
```
定义了第一个枚举的值后，后续的枚举成员会自增。

### 自定义枚举

```ts
enum Color {
  Red = 1, // 1
  Green = 6, // 6
  Blue = 10 // 10
}
```
可以自定义所有的枚举值。
