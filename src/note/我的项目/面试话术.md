## 自我介绍

你好，我叫王寒，毕业于东华理工大学软件工程专业，目前已经有6年的前端开发经验，技术上以 Vue 生态和 TypeScript 为主，对 Vue2、Vue3、工程化、性能优化、SSR、微前端都有比较深入的实践，也有 React、Node.js 和部分全栈开发经验。我的项目经历比较完整，既做过直接面向用户的业务系统，也做过偏平台型和基础设施型的项目。

在 BT 教育平台这个项目里，我负责过前端整体架构设计、技术管理、CI/CD、发布部署和性能优化，比较有代表性的成果是搭建官网 SSR 体系，把首屏加载时间从 2.1 秒优化到 0.5 秒；同时我也主导了公司级微前端架构升级，基于 qiankun 在 3 个月内整合了 10 多个不同技术栈的历史项目，解决了样式冲突、路由冲突、状态同步和部署耦合等问题。

另外我还从 0 到 1 设计过低代码平台，负责编排引擎、渲染引擎、变量系统和拖拽引擎这几块，重点解决复杂嵌套布局下的拖拽放置算法和复杂页面渲染性能问题。还有一个我比较想重点讲的是前端全链路监控平台，我主导设计了轻量级监控 SDK 和可视化分析平台，把前端错误、性能、行为轨迹、告警和 SourceMap 排障打通，覆盖了 10 多个核心业务，故障发现时效从用户反馈的小时级缩短到了系统告警的分钟级。

除了项目本身，我也比较关注研发效能和 AI 工具落地，平时会结合 Codex、Cursor、Claude Code 这些工具去优化需求拆解、Code Review、测试增强和工程交付效率。所以我现在更希望加入一个既重视业务结果、也重视技术质量和长期演进的团队，在前端业务开发之外，继续发挥我在架构、稳定性和协同推进方面的价值。”
