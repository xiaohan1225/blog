- shift + enter 换行
- Ctrl + G 用 vscode 来编辑
- 按 ! 键执行 bash 脚本
- open index.html 用浏览器打开文件

可以任意执行终端命令，不需要询问用户。
```bash
claude --dangerously-skip-permissions
```

- /tasks 查看后台任务
- /compact 压缩上下文
- /clear 清除上下文
- 使用 /init 初始化 CLAUDE.md 文件， 修改 CLAUDE.md 文件后，先推出，再按 claude -c 重新进入，新的 CLAUDE.md 文件就能起作用了。
- /memory 打开 CLAUDE.md 文件
- 回滚
  - /rewind
  - 连续按两次 ESC 进入回滚页面，每次输入请求的时候，claude code 都会创建一个回滚点
  - claude code 只能回滚自己写入的那些文件，而那边用终端命令生成的文件，claude code 没办法回滚，要做到精确回滚，还是适合用 git


## 传入图片
1. 拖动图片到终端
2. 复制图片，按 ctrl + v，即使是 mac 电脑

## 恢复会话
- /resume 历史会话列表
- claude -c 打开 claude code，并自动恢复上一次的对话
