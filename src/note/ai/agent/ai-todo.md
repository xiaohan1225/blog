用 Cursor Automation 搞了几个自动化脚本，现在每周基本不用管琐事了。

第一个是周报自动整理，每周自动从各个分支收集 commit 信息，按项目分类整理生成周报并直接提 PR，我只需要 Review 一下，再也不用手动复制粘贴了。

第二个是项目健康检查，每周对我的主项目跑 SEO audit 和最佳实践扫描，结果直接发到 Slack，看一眼就知道有没有问题，需要修复的再动手。

第三个直接监控项目的 Sentry，一有异常就开分支修复并提 PR，连 PR 的描述和标题都自动写好，我只要合代码就行。

第四个是前端周刊自动化，每周从 GitHub Trending、Twitter、掘金、Medium 等渠道收集前端圈动态，自动去重分类，整理成周刊发布到博客，再也不用自己到处刷信息流了。
